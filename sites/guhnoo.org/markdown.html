Markdown Extensions

werchan posts support a number of extensions to Markdown:

  • Smart fractions
  • LaTeX-style dash parsing
  • Intra-word emphasis supression
  • Tables
  • Fenced code
  • Autolink
  • Strikethrough
  • Space headers
  • Hard line break
  • No empty line before block
